This page will help you clear browser cache and reload the application to fix the infinite refresh issue.

Manual Steps (if buttons don't work):

  1. Open Developer Tools (F12)
  2. Right-click the refresh button
  3. Select "Empty Cache and Hard Reload"
  4. Or press Ctrl+Shift+R (Cmd+Shift+R on Mac)
